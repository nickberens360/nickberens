I wanted my new site to have content sections instead of pages. I wanted those sections to take up the full height and width of the browser window.

Aaaand, I also wanted the content to be centered horizonally and vertically. After some tinkering and lifting code from others, this is what I've come up with.

The 100% height bit

At first I came up with a jQuery solution that detected the height of the window and then applied it to the height of the desired container. This of course is stupid because it can be done with CSS.

I'm not going to walk through what each part of the code does, dissect the code samples to get an idea. It's pretty straight forward.

See the Pen zwumv by Nick Berens (@nickberens360) on CodePen

The vertical centering stuff

The following code is responsible for vertically and horizontally centering the content. The idea is to make the containing element behave like a table and to then make the container with the content behave like a table cell. 


.table{
	display:table;
	width:100%;
	overflow: hidden;
	position:relative;
}
.cell {
	display: table-cell;
	vertical-align: middle;
	width: 100%;
	margin: 0 auto;
	text-align: center;
}

The end result

Combining the above techniques gave me just what I was looking for, containers that are 100% the height of the browser window and content that is vertically and horizontally centered. You can certainly combine the .full-height and .table classes to be less verbose if you'd like, but that's about it.
